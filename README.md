# Clinicaxd

Landing y prototipo de una clinica digital con secciones de especialidades, reservas rapidas, testimonios y blog informativo. Pensado para mostrar la experiencia completa de pacientes y personal medico.

## Tecnologias
- Next.js y React
- Tailwind CSS con componentes shadcn/ui
- TypeScript

## Requisitos
- Node.js 18 o superior
- npm (incluido con Node)

## Puesta en marcha local
1. Clona el repositorio: git clone <tu-usuario>/<repo>.git
2. Entra a la carpeta del proyecto: cd clinicaxd
3. Instala dependencias: npm install
4. Levanta el entorno de desarrollo: npm run dev
5. Abre http://localhost:3000 en el navegador.

## Scripts utiles
- npm run dev: inicia el modo desarrollo con recarga en vivo.
- npm run build: genera la version optimizada para produccion.
- npm start: sirve la compilacion generada por el build.
- npm run lint: ejecuta ESLint sobre el codigo del proyecto.
